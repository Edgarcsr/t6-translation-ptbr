# Product

## Register

product

## Platform

web

## Users

Portuguese-BR speakers playing Call of Duty: Black Ops II on Plutonium who need a simple way to download and install the fan translation. Secondary audience: translation contributors who deploy or test their work.

## Product Purpose

A desktop companion app for the T6 PT-BR translation project. It downloads the latest translation release from GitHub and applies it directly to the Plutonium game folder — no manual file copying or mod activation required. Success means a player can go from zero to a fully translated game in two clicks.

## Positioning

The simplest way to install and manage the BO2 PT-BR translation — download, apply, and play, all from one clean interface.

## Brand Personality

Gamer, practical, clean. The interface is dark and no-nonsense — it gets the job done without flashy distractions. Every element serves the workflow.

## Anti-references

- Overly gamified UIs with heavy textures or skeuomorphic elements (Call of Duty menu clones)
- Busy dashboards with unnecessary data visualizations for a single-purpose tool
- Light/near-white themes — this is a gaming tool, not a productivity SaaS

## Design Principles

- **One job, well done.** Every pixel serves the download → apply → play flow. No feature creep.
- **Status clarity first.** The user must always know what's happening (idle, downloading, applying, done, or error) without hunting for it.
- **Dark and deliberate.** Dark theme isn't a default — it's the right call for a gaming-adjacent desktop tool used in low-light environments.
- **Resilient by design.** Network failures, missing files, and bad releases are expected. Handle each gracefully with clear recovery paths.

## Accessibility & Inclusion

- WCAG AA contrast minimums for all text and interactive elements
- Status conveyed through both color AND text labels (not color alone)
- Keyboard-navigable settings panel with Escape-to-close
- Reduced-motion support for all animations
